CSS Zen garden

In the dark ages of web-design, we were very limited in which styles we could play around with. CSS was the solution, and became very popular! It is however less relevant for functionality. JavaScript connects to HTML elements, but does not "need" CSS.

Rather advanced, but below you can see how the colours for this page — in two different modes — are set.

            
            /* All the variations var(); listed here */
            :root {
                --colour-body: #e5e7f0;
                --colour-font-primary: #000000;
                --colour-font-secondary: #4a4cad;
                --colour-font-auxillary: #333333;
                }

            @media (prefers-color-scheme: dark) {
            :root {
                --colour-body: #522121;
                --colour-font-primary: #f9f9f9;
                --colour-font-secondary: #a8a9d8;
                --colour-font-auxillary: #f9ddf9;
                }
            }
