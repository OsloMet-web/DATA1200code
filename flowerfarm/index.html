Welcome to Flower Farm Follo

And welcome to the page "index.html"! When a link points to a folder (also known as a directory), the browser will look for "index.html". Without an index.html, any link or web address must point to a specific file, like https://www.oslomet.no/welcome.html.

This is a general paragraph. The second one, in fact. As you can see, when we define a paragraph, the browser will by default add space above or below. Now, it is not a very nicely formated paragraph, running all the way from edge to edge of the browser. Therefor, in the code, remove the comment that says "Un-comment me" in the <head>. (It's the "link" for "spacing.css") See more below.

For an example of the absolute simplest html you can expect a browser to display, go to simplest.html.

  <html>
  <body>
  Hello world!
  </body>
  </html>
...that is all you need — but correct HTML needs more!

About this page

This page has a CSS-sheet that removes the default margin, so there is no space between the left-most edge of the browser and the text.

The main.css sets the 'body' to 'margin: 0'. While spacing.css sets the 'main' to 'margin-left: 2rem;' — that link has been commented out. Therefore, it does not apply. Doesn't look good, does it? But if you take this code and remove the commenting-out, spacing.css will apply and the page looks much better.

Alternatively, simply go to "The Farm"-page. (Menu at top.) That page has both stylesheets
