Tags, elements, properties

All the <p> and <h2> "code" here are tags. Usually, there is an opening and a closing tag. Taken together, it is an element. This text is written inside a paragraph-element, a set of <p>–</p> tags. An element can have a property. For example, the <img>-tag above, has a <src>-property with a path for the image.

Selectors in CSS

In the stylesheet (or multiple stylesheets), elements can be selectors.
For example , inside the body of this document, there are three sections defined by (semantic) elements: 'header', 'body', and 'footer'. These can be used in CSS to select which elements to style, i.e. thy can be used as selectors. In this page, in the stylesheet "spacing.css", I have used 'main' as a selector to add margins to the image at text, but not the menue on top – which is inside the 'header' element. (At the bootom of the page is a paragraph outside the 'main' element.

Learning elements

First of all, note the images. The first one is as simple code as allowed. <img>-element and alt-text only. No scaling! (Re-size your browser window.) The second one is the simple alternative. The picture-element allows for example switching from a large image to a smaller one. There are more advanced options for scaling pictures, but begin here.

Second, in this page supports the reader-mode. Not big deal, but helpful. Semantic structure is necessary for the reader-mode to trigger.
