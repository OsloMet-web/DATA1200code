The only difference between these two boxes is "box-sizing". Both have these CSS-properties:

        div {
            width: 500px;     <== This is the interesting property + value pair
            padding: 20px;
            margin: 10px;
            border: 8px solid rgb(160, 92, 92);
            background: rgb(255, 255, 175);
            }

This is a content-box, the default.

TOTAL width =
'width' + 'padding' (inside) + border (red line).
=> total width (outside border) > CSS-width.

 

This is a border-box, not a default.

TOTAL width =
'width', including 'padding' (inside) & border (red line).
=> total width (outside border) = CSS-width.
